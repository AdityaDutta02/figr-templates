Rule 2
Space groups, not just
elements.
20
20
Unrelated items sit far
Note:
Equal gaps flatten hierarchy. Different
gaps create meaning.
@figr.design
Rule 3
Give breathing room before
headings, not after.
More space above
a heading than
below
Note:
Space above sets context. Space below
connects content.
@figr.design
Rule 4
Padding > margin for
components.
Padding controls internal
breathing room
Margin controls
external distance
Note:
Padding shapes the component. Margin
defines relationships.
@figr.design
Rule 5
Vertical rhythm is invisible
glue.
3x
2x
2x
3x
Sections that
breathe at the
same intervals
Note:
If rhythm breaks, the layout feels off even
if you can't explain why.
@figr.design
Rule 6
Tight spacing = importance.
Spacious =
premium, editorial
Note:
Less space pulls focus. More space
signals separation.
@figr.design
Rule 7
Test spacing at actual
screen size.
1920px
1024px
479px
100% zoom only
Note:
If you don't test at 100%, you're designing illusions.
